Dynamic Field Helper

Transform static help text into smart, context-aware insights. Defined in Python, rendered in HTML, displayed on demand.

Why Dynamic Help?

Standard Odoo help="..." attributes are static. They can't tell you the current customer's credit limit, the stock level of a product, or warnings based on other fields. Dynamic Field Helper changes that.

Python Logic

Define your help content using standard Python methods. Access self, search other models, or perform calculations.

Rich HTML

Don't limit yourself to plain text. Use Bold, Italics, Lists, Colors, and Tables to convey information clearly.

Hover Interaction

A sleek popover appears instantly on hover. Designed to look native to Odoo 17/18 web interface.

Simple Implementation

Just add one attribute.

No XML modifications required. No complex widget registrations. Just point the helper attribute to your method.

  • Clean Syntax
  • Zero Database Storage
  • Hot-reload capable
field_name = fields.Char( string="My Field", helper="_get_help_content" ) def _get_help_content(self): return f""" <b>Balance:</b> {self.balance}<br/> <i>Generated at {datetime.now()}</i> """
